# NYC-rapes
[Lede Project 1] Reported Rapes in NYC: Trends To Be Aware of in 2026
Data Project Assignment 1

OVERVIEW

This is my submission for the first data project of the 2026 LEDE Program. 

The data used in this project was the NYPD Complaint Data Current (Year-to-Date), downloaded as a .csv file from the NYC Open Data website. The data lists crimes reported to the New York Police Department in 2026 (year to date). My goal was to analyze the data specific to rape cases.


DATA ANALYSIS

With guidance from TA Anastasiia Morozova, I used pandas in VSCode to clean the data with the goal of finding the number of rape reports for 2026, their distribution by borough and precinct area, common locations for the incidents, and, finally, the most common time periods in which rapes reportedly happened. Then, percentages and number of crimes per capita were calculated. Data from reputable organizations, such as the UN, the Department of Justice, and RAINN, were also used in the copy to provide further context for the NYC data and to compare it to national averages. 


DATA VISUALIZATION

For the data visualization part of the project, I used Datawrapper to create four visuals: 

- A chloropleth map of NYC’s precinct areas, showing the top regions accounting for most rape reports; 

- A range plot showing the number of rapes per capita by borough to put the numbers into perspective (boroughs have vastly different population sizes); 

- A stacked column chart showing the most frequent locations in which reported rapes took place: residences, motel/hotel/street, etc;

- A heat map table showing the time periods in which reported rapes happened;

FINDINGS

The data shows that The Bronx accounts for most reported rapes in 2026 and a rape crime per capita twice as high as Brooklyn, which is second in number of rape reports. In addition, the data shows that depending on the borough, the majority of rapes occur at different times of the day. The crime predominantly happens in a residence.
